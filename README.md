# RAG-LlamaIndex

## Setup
1. Clone the repository:
```bash
git clone <repository_url>
cd RAG-LlamaIndex
```

2. Create and activate a virtual environment (on WSL):
```bash
python3 -m venv .venv
source .venv/bin/activate
```

3. Install the required dependencies:
```bash
pip install -r requirements.txt
```
