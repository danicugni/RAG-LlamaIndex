# RAG-LlamaIndex

## Setup
1. Clone the repository:
```bash
git clone <repository_url>
cd RAG-LlamaIndex
```

2. Create and activate a virtual environment:
```bash
py -3.10 -m venv .venv
.venv\Scripts\activate  # On Windows
```

3. Install the required dependencies:
```bash
pip install -r requirements.txt
```
